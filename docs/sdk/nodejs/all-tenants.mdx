# Node.js All Tenants Check Examples

### 1. Perform All Tenants Check

The all-tenants check allows you to verify if a specific user is permitted to perform a certain action on a resource across all tenants. The request is a GET request to the `/allowed/all-tenants` endpoint.

- **user**: The identifier of the user for whom you want to check permissions.
- **action**: The action to check permission for (e.g., read, write, delete).
- **resource**: The resource type on which the action is being performed (e.g., document, folder).
- **context**: Additional context information for the permission check (optional).
- **sdk**: The SDK language being used (e.g., node, python, java).

```javascript
const axios = require('axios');

axios.get('http://localhost:7766/allowed/all-tenants', {
  params: {
    user: 'employee1',
    action: 'read',
    resource: 'document',
    context: {},
    sdk: 'node'
  },
  headers: {
    Authorization: 'Bearer API_SECRET_KEY'
  }
})
.then(response => {
  console.log('Permission check result:', response.data);
})
.catch(error => {
  console.error('Error performing all-tenants check:', error.response.data);
});
